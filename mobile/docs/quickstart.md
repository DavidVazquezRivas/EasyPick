
# Quickstart

Como estamos usando una version antigua de expo, es necesario instalar una version compatible de react y react-native. Para esto, se han instalado las siguientes versiones:

- react: 19.1.0
- react-native: 0.81.5
- expo: 54.0.2

Para instalar lo anterior, ejecutamos el siguiente comando en la terminal, dentro de la carpeta del proyecto:

```bash
npm install
```

finalmente, tambien tendremos que instalar la dependencia de status bar, para esto ejecutamos el siguiente comando:

```bash
npm install expo-status-bar
```

Con esto, ya tenemos todas las dependencias necesarias para ejecutar el proyecto. Ahora, para iniciar el proyecto, ejecutamos el siguiente comando:

```bash
npx expo start
```

Esto iniciará el servidor de desarrollo de expo, y nos mostrará un código QR en la terminal. Para abrir la aplicación en nuestro dispositivo móvil, escaneamos el código QR con la cámara de nuestro dispositivo, y se abrirá la aplicación en nuestro dispositivo. Necesitamos tener la aplicación de expo instalada en nuestro dispositivo para poder abrir la aplicación. Si no la tenemos instalada, podemos descargarla desde la tienda de aplicaciones de nuestro dispositivo (Expo Go para Android o Expo Go para iOS).
